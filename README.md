# MuTube

MuTube (Music Tube)

## 간단한 설명

    음악 영상만 전문적으로 공유하는 웹

## 사용한 기술

    VanillaJS , nodeJS express
    MVC design pattern( model, view, controller )
    Express template PUG
    MongoDB - Mongoose
    dotenv( 환경변수 )
    Multer ( nodeJS 미들웨어 , 업로드한 파일 받아 처리 )

## 기능

    - [x] 회원가입
    - [x] 로그인
    - [o] 비디오 업로드
    - [o] 비디오 상세보기
    - [o] 비디오 수정
    - [o] 비디오 삭제
    - [x] 회원정보 수정
    - [x] 깃허브, 지메일로 가입 및 로그인 기능
